# Death Note FanWiki

Fan-made wiki website about the anime Death Note, created as my first website using Bootstrap.

## Technologies
- HTML5
- CSS3
- Bootstrap

## Purpose
This project was created to practice building responsive layouts and organizing content using Bootstrap.

## What I learned
- How to structure multi-section web pages
- How to use Bootstrap grid system
- How to create responsive designs
